Форк штатного приложения Fairphone 2, модифицированный для работы на других устройствах.

Fairphone Checkup позволяет проверить различные компоненты вашего телефона:

- Экран
- Динамики
- Микрофоны
- Мобильное подключение
- Wi-Fi
- Функцию вибрации

Это может быть полезным для диагностики проблем с вашим устройством.
